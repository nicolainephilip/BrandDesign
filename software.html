Tools I Use

  • Adobe Illustrator

  • My main tool for logo design and vector illustrations. It's perfect for clean, scalable, and professional designs.

  • Adobe Photoshop

  • Used for image editing, mockups, and layout refinement. Ideal for crafting visuals that need depth, texture, and photo-based work.

  • Adobe Express

  • A quick and easy tool for social media graphics and lightweight design tasks. Great for turning ideas into visuals fast.

  • CorelDRAW

  • Powerful for vector-based work and precision layout tasks. I use it on select projects that require detailed print and layout work.

  • Canva

  • A client-friendly platform for editable designs like social media templates and basic branding kits. Great for businesses that want to reuse templates with ease.

  • Affinity Designer

  • An alternative vector tool I use for some print and logo designs. It offers flexibility and powerful features for creative work.

  • Sketch

  • Used mainly for digital interface mockups and web layout design. Great for planning website structure and UI components.

  • Adobe InDesign

  • Perfect for designing multi-page layouts like brand books, flyers, and business cards. It’s my go-to for print-ready materials.

  • Procreate (iPad)

  • Used for sketching, hand-drawn elements, and concept development. It’s perfect for adding a personal and artistic touch to projects.

  • Figma

  • I use Figma for website wireframes, UI/UX design, and prototypes. It's great for collaboration and creating responsive web designs.
